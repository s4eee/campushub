# CampusHub

A centralized platform for discovering events, workshops, hackathons, competitions, seminars, and opportunities happening across a college campus.

## Problem

College event information is often scattered across multiple channels such as WhatsApp groups, Instagram posts, stories, emails, and committee announcements. Students must constantly switch between platforms to stay informed about upcoming opportunities.

CampusHub aims to solve this problem by providing a single dashboard where students can view all campus events in one place.

## Features

### Student Features

- View upcoming events
- Search and filter events
- Event details page
- Save/bookmark events
- Calendar view
- Category-based discovery

### Organizer Features

- Create and publish events
- Upload event posters
- Add registration links
- Edit and manage events

### Event Categories

- Workshops
- Hackathons
- Technical Events
- Cultural Events
- Sports Events
- Guest Lectures
- Committee Recruitments
- Competitions

## Project Goal

CampusHub serves as a centralized event discovery platform for students, reducing information fragmentation and making it easier to stay updated with opportunities across campus.
